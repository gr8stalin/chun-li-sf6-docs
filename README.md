# Street Fighter 6 Notes for Chun-Li

In an attempt to both move my various game notes off of OneNote and make them easier to share, I've taken all of my personal notes for Chun during my time playing Street Fighter 6 and created a Github Pages site using the Jekyll theme `just-the-docs`.

These notes are more for competitive SF6 players who have a decent understanding of the way fighting games are played.

## Building and previewing your site locally

Assuming Jekyll and Bundler are installed on your computer:

1.  Change your working directory to the root directory of your site.

2.  Run `bundle install`.

3.  Run `bundle exec jekyll serve` to build your site and preview it at `localhost:4000`.
